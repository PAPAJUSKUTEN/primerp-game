# 🛒 Zaawansowany Sklep Internetowy - Full-Stack App

Profesjonalna i dopracowana aplikacja sklepu internetowego oferującego produkty cyfrowe oraz usługi biznesowe. Projekt łączy nowoczesny front-end z bezpieczną walidacją danych po stronie serwera (back-end).

## 🚀 Główne Funkcje

- **Dynamiczny Front-end:** Asynchroniczne pobieranie danych o produktach bez przeładowywania strony (`Fetch API`).
- **Koszyk w czasie rzeczywistym:** Dynamiczne przeliczanie sumy i ilości produktów w przeglądarce.
- **Bezpieczny Back-end:** Weryfikacja cen i poprawności koszyka bezpośrednio na serwerze (ochrona przed manipulacją cen w kodzie źródłowym strony przez klienta).
- **Architektura REST API:** Czytelne punkty końcowe (`GET /api/products`, `POST /api/checkout`).

## 🛠️ Technologia

- **Kliencka (Front-end):** HTML5, nowoczesny CSS3 (zmienne CSS, CSS Grid, Responsive Web Design), JavaScript (ES6+).
- **Serwerowa (Back-end):** Node.js, Express.js.
